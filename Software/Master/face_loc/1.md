# Smart Mirror Face Tracking Skill

使用这个仓库根目录的 skill 来管理本地人脸跟踪服务。

## Actions

### `start_face_tracking`

启动后台服务：

```bash
python scripts/face_tracking_skill.py start_face_tracking --config configs/default.yaml --source-type camera --camera-index 0 --headless
```

可选参数：

- `--source-type camera|rtsp|video_file|capture`
- `--camera-index 0`
- `--rtsp-url rtsp://...`
- `--video-path /absolute/path/demo.mp4`
- `--model-backend opencv_yunet|insightface_onnx|insightface_faceanalysis`
- `--model-path weights/face_detection_yunet_2023mar.onnx`
- `--model-name <pack-or-model-name>`
- `--device cpu|auto|cuda`
- `--show-gui` 或 `--headless`
- `--host 127.0.0.1 --port 8000`

### `get_face_tracking_result`

获取最近一次检测结果：

```bash
python scripts/face_tracking_skill.py get_face_tracking_result
```

### `get_face_tracking_status`

获取服务状态：

```bash
python scripts/face_tracking_skill.py get_face_tracking_status
```

### `stop_face_tracking`

停止后台服务：

```bash
python scripts/face_tracking_skill.py stop_face_tracking
```

## Main Entry

本地直接看效果：

```bash
python main.py --config configs/default.yaml
```

无窗口模式：

```bash
python main.py --config configs/default.yaml --headless
```

## Files

- `main.py`: 本地直接启动入口
- `scripts/face_tracking_skill.py`: skill action wrapper
- `actions.schema.json`: action 输入输出定义
- `skill.manifest.json`: 工程级元数据
